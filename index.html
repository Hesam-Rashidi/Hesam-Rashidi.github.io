Production-level dynamic parking pricing for the City of Toronto. Learns a mapping from real-time meter transactions to occupancy and adjusts rates by location/time to balance availability and turnover.

Winter Maintenance Optimization Tool

Web-based decision support for municipal snow plowing. Runs arc-routing on road networks with service priorities and turn costs, featuring scenario testing, boundary adjustments, and KPI comparisons before deployment.

Traffic-Aware Cost Matrices for TSP Without O(n²) Overhead

An API that constructs a traffic-aware cost matrix for the Travelling Salesman Problem by retrieving only the edges most likely to appear in the optimal tour, avoiding the need for a full O(n²) matrix. This reduces query delays and significantly lowers routing API costs.
